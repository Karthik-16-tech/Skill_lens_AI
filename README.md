# Skill Lens AI

An intelligent AI-powered career and decision simulation platform that helps you predict your future outcomes and make informed life decisions.

## 📋 About

**Skill Lens AI** is a sophisticated platform that uses artificial intelligence to simulate your future decisions and predict their outcomes. Whether you're considering a career change, evaluating life choices, or planning your trajectory, Skill Lens AI provides data-driven insights and visualizations to help guide your decisions.

### Key Features

- **AI Decision Simulation** - Run intelligent simulations on your life choices and see predicted outcomes before committing
- **Career Path Predictor** - Compare career trajectories with AI-driven market analysis and growth forecasts
- **Future Outcome Visualization** - See your possible futures visualized across income, satisfaction, and stability metrics
- **Mind Map Generator** - Auto-generate decision trees that map every possibility and its long-term impact
- **Stress & Income Predictions** - Get personalized stress levels and income projections for each career path
- **AI Recommendation Engine** - Receive actionable suggestions backed by real market data and trend analysis

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher) - [Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- npm or bun package manager

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Karthik-16-tech/Skill_lens_AI.git
cd Skill_lens_AI
```

2. **Install dependencies:**
```bash
npm install
# or if using bun
bun install
```

3. **Start the development server:**
```bash
npm run dev
# or
bun run dev
```

The application will be available at `http://localhost:5173`

## 🏗️ Tech Stack

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Form Management**: React Hook Form
- **State Management**: TanStack React Query
- **Routing**: React Router
- **Animations**: Framer Motion
- **Testing**: Vitest
